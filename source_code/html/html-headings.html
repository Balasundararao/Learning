Elements

Block Level Elements

Block elements appear on the screen as if they have a line break before and after them. For example the P, H1, H2, H3, H4, H5, H6, UL, OL, DL, PRE, HR, BLOCKQUOTE, and ADDRESS elements are all block level elements. They all start on their own new line, and anything that follows them appears on its own new line.

Inline Elements

Block elements appear on the screen as if they have a line break before and after them. For example the b, i, u, em, strong, sup, sub, big, small, li, ins, del, code, cite, dfn, kbd, and var elements are all block level elements. They all start on their own new line, and anything that follows them appears on its own new line.
